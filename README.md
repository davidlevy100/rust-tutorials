### [neetcode/](neetcode/)

Practicing the [NeetCode 150](https://neetcode.io/practice/practice/neetcode150) in Rust. All 27 Easy problems are scaffolded — one standalone Cargo project per problem, grouped by category, each a `todo!()` stub in `src/lib.rs` with the example cases in `tests/`. Implement the body, then `cargo test` in that folder. See [neetcode/README.md](neetcode/README.md).
